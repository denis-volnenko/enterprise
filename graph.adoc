= Структуры JSON-объектов

== Плоские структуры

*ProjectPlainDto*
----
{
  "id": "1",
  "name": "Демо проект"
}
----

*TaskPlainDto*
----
{
  "id": "1",
  "name": "Демо задача",
  "projectId": "1"
}
----

== Граф (нециклический, направленный, полный, валидный)

ProjectGraphDto
----
{
  "id": "1",
  "name": "Демо проект",
  "tasks": [{
    "id": "1",
    "name": "Первая задача"
  },{
    "id": "2",
    "name": "Вторая задача"
  }]
}
----

== Граф (нециклический, направленный, облегченный, валидный)

ProjectGraphDto
----
{
  "id": "1",
  "name": "Демо проект",
  "tasks": ["1", "2"]
}
----
