= Структуры JSON-объектов

== Плоские структуры

*ProjectPlainDto*
----
{
  "id": "1",
  "name": "Демо проект"
}
----

*TaskPlainDto*
----
{
  "id": "1",
  "name": "Демо задача",
  "projectId": "1"
}
----

== Нециклический направленный граф (DAG)

ProjectGraphDto
----
{
  "id": "1",
  "name": "Демо проект",
  "tasks": [{
    "id": "1",
    "name": "Первая задача"
  },{
    "id": "2",
    "name": "Вторая задача"
  }]
}
----
